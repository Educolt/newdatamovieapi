## 🗂 Como instalar o projeto

```bash
  # Clonar o repositório
  $ git clone https://github.com/Educolt/newdatamovieapi

  # Entrar no diretório
  $ cd newdatamovieapi

  # Instalar as dependencias
  $ yarn

  # Inicia o projeto
  $ yarn dev

```
